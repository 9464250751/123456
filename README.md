# 123456 - Structured Data Assignment

A C# Windows Forms application demonstrating the use of structs and data structures.

## Project Description
This project implements various C# structs including Vehicle, Person, Pet, SmartPhone, PlayingCard, and DeckOfCards, along with a Windows Forms interface for interacting with these data structures.

## Building the Project
```bash
cd StructuredDataAssignment
dotnet build
```

## Running the Application
```bash
cd StructuredDataAssignment
dotnet run
```

## Project Structure
- `StructuredDataAssignment/` - Main Windows Forms application
- `StructuredDataAssignment.Tests/` - Unit tests for struct implementations

## Requirements
- .NET 10.0 SDK
- Windows Desktop runtime (for running the Windows Forms application)

See `StructuredDataAssignment/IMPLEMENTATION_SUMMARY.md` for detailed implementation information.
